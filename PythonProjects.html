I've been using Python for the same thing everyone else is using Python, AI. I've learned a lot about AI from developing my own neural nets in C++ and tinkering with python libraries. While working at Oracle I participated in a few "learn-a-thons" where we learned and played with different types of AI models. Below are some of my projects I've worked on in Python.


Maze AI

A Python AI to solve mazes. it uses BFS and DFS to find the correct route. It can also output images of the path it took to solve the maze.

The image to the left is an image of the path it took to complete a maze using BFS.

Student Registration

A Django Python project to display students that are registered for classes given the user selected semester. We created a webscraper in Python to scrape the student information off of the Banner webpage and stored that data in an accessible database. This was used for staff / instructors to view how many students were registered in their classes. The project no-longer works because we shut down the webscraper and database containing the scraped data.

Diffie Hellman Key Exchange

I created an encryption class that will create and complete a Diffie Hellman Key Exchange. My plan is to one day finish this with an End-To-End encryption chat program.
