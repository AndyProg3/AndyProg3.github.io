Chaparral Part Search

This is a webpage for verified boat contractors to search and order boat parts. This was created in PHP using Smarty3 for templating. MYSQL was used for the back end connecting to already existing database and adding new tables for the ordering system.

Robalo Boats

In my freelance work, I was requested to redesign the Robalo Boats website. They gave me a pdf with some design ideas to follow with. I used a combination of PHP, MYSQL and Smarty3 to complete the design. You can see the finished website at robalo.com

Login System
