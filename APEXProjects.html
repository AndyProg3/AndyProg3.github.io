JSON Templates

We had a need to know when a 3rd party would change their JSON response that we consume. So I created a set of PL/SQL pacakges and tables to compare a "perfect" version that we expected to all incoming JSON responses. This tool allowed for different types of alerting and was very configurable for any JSON (Not just our specific JSON responses)
